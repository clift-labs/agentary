# Global Todos

These are tasks that cut across all projects - personal items, recurring duties, and cross-project work.

## Format

```markdown
- [ ] Task description @due(YYYY-MM-DD) #tag
- [x] Completed task
- [/] In progress task
```

## Priority Levels

- `#urgent` - Do today
- `#important` - Do this week
- `#someday` - Nice to have

## Recurring Items

Use `@recurring(daily|weekly|monthly)` for repeating tasks.
