# Projects

Each project is a first-class citizen with its own folder containing:
- `.socks.md` - Project-specific context
- `notes/` - Project notes
- `todos/` - Project-specific tasks
- `research/` - Research topics and findings

## Naming Conventions

- Use lowercase with hyphens: `my-project-name`
- Be descriptive but concise
- Avoid special characters

## Project Lifecycle

1. Create with `dobbie project new <name>`
2. Switch with `dobbie project switch <name>`
3. Active project is remembered between sessions
