# Global Schedule

Time-blocked events and appointments that span all projects.

## Format

```markdown
## YYYY-MM-DD

- 09:00 - 10:00 | Meeting with team #meeting
- 14:00 - 15:00 | Focus time #focus
```

## Event Types

- `#meeting` - Meetings and calls
- `#focus` - Deep work blocks
- `#admin` - Administrative tasks
- `#personal` - Personal appointments
